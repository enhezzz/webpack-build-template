##  vue ssr的webapck模板

>   根据 https://ssr.vuejs.org/ 做的一个配置模板
>
>   运行npm run start即可运行该demo    
>
>   npm run build-server-bundle 构建服务器端server-bundle.json
>   
>   npm run build-client-bundle 构建客户端端bundle

>   目录结构
*   build(webpack构建配置)

*   server(服务器和构建生成文件代码)

*   src(vue ssr配置以及入口文件)
